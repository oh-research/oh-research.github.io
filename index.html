대포 실험실

🔫

발사 시간과 거리의 관계를 직접 관찰하면서 데이터와 좌표평면을 함께 보는 실험입니다.

  • 스페이스바 또는 숫자 입력으로 발사
  • 데이터 테이블과 그래프를 동시에 확인
  • 교사 설정으로 선형, 2차, 제곱근 관계식 전환

Pattern Fitting

규칙 맞추기 실험실

📈

좌표 데이터를 보고 1차식 또는 2차식 규칙을 추정해보는 피팅 실험입니다.

  • 점 데이터를 수동 입력하거나 URL 파라미터로 전달
  • 오차 점수와 시행 기록으로 추정 과정 추적
  • 힌트와 잠금 옵션을 활용한 수업용 운용 가능
실험 시작 새 탭에서 열기

로컬에서 바로 확인하려면 이 폴더에서 python3 -m http.server 8000 실행 후 http://127.0.0.1:8000 로 접속하면 됩니다.

각 실험 페이지 상단에는 다시 이 홈으로 돌아오는 링크도 추가했습니다.
